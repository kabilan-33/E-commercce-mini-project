 e-commerce website built as a mini project using HTML, CSS, and Bootstrap. It showcases product categories like Fashion, Electronics, and Jewellery, and includes a payment and confirmation page for a complete front-end experience.

🚀 Features
📦 Product category pages: Fashion, Electronics, Jewellery

🧭 Navigation bar for smooth browsing across pages

🖼️ Product cards with pricing and Buy Now buttons

💳 Payment form with input validation

✅ Confirmation page after successful order

🛠️ Technologies Used
HTML5

CSS3

Bootstrap 4.5

📁 Pages Included
Page	Description
index.html	Home page with quick category access
fashion.html	Fashion products like shirts & skirts
electronic.html	Electronics like laptops & mobiles
jewellery.html	Jewellery items like jhumkas & kangans
about.html	About the website
contact.html	Contact details
payment.html	Secure payment form
confirmation.html	Order success confirmation

📌 How to Run
Download or clone the repository

Open index.html in your browser

Navigate through product categories

Click Buy Now to go to payment

Submit payment form to see confirmation

💡 You can also use VS Code + Live Server for best experience.
